Zeecurity Live – Smart Security App

Zeecurity Live is a full-stack smart residential security management system built using the MERN stack. It helps manage visitor entries, complaints, SOS alerts, payments, notices, and communication through role-based dashboards for Admin, Guard, and Resident.

Features
Role-based dashboards
Visitor management
Complaint tracking
SOS emergency alerts
Payment management
Notices and communication
Tech Stack

React.js, Node.js, Express.js, MongoDB, Material UI
